POCKETSYNTH GUIDE

Start making a track.

1

Choose a sound

Tap the colored controls for Engine, Scale, FX, and Tempo. The screen always shows the active setup.

2

Select a layer

Choose any layer in the row, or press numpad 1–9. The selected layer is the only layer that REC or ERASE changes.

3

Build it up

Loop Layer 1, select Layer 2, then record. Layer 1 keeps playing while only your new Layer 2 notes are captured.

4

Set your range

Tap OCTAVE to cycle C2–C4, C3–C5, C4–C6, and C5–C7. The same computer-key layout moves with the selected range.

5

Play the keyboard

Keys run left to right. Use Z–M for the lower octave and Q–U for the upper octave. Glowing keys belong to the selected scale.

This is an original PocketSynth interface, informed by portable instrument workflows—not a replica of any third-party device.
